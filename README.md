# 🏥 نظام إدارة العيادة الطبية

نظام ويب متكامل لإدارة عيادة طبية، يشمل:
- حجز وإدارة المواعيد.
- ملفات المرضى (بيانات، زيارات، وصفات طبية).
- كشف حساب المريض وإدارة الخزينة.
- صلاحيات مستخدمين (طبيب - سكرتارية).

## 🚀 تشغيل التطبيق
- قم بفتح `index.html` مباشرة في المتصفح.
- أو استخدم [GitHub Pages](https://اسم_المستخدم.github.io/اسم_المستودع).

## 🔧 التقنيات المستخدمة
- HTML + CSS + JavaScript (Pure)
- Firebase (Authentication + Firestore)
- مكتبة XLSX لتصدير Excel

## 👥 صلاحيات المستخدمين
- **الطبيب**: صلاحية كاملة (إدارة كل شيء).
- **السكرتارية**: صلاحيات محددة (حسب التكوين).

## 📦 الاعتماديات
يتم تحميلها تلقائياً من CDN:
- Firebase (v10)
- XLSX (v0.18)
